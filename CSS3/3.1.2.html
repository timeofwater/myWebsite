CSS的基本语句格式为:
        选择器{属性:属性值;}    选择器:通常是需要定义的HTML元素或标签
                                属性:元素属性

    p {
        text-indent: 2em;
    }

    组合选择器:
        选择器,选择器{属性:属性值;}

    a,p {
        text-indent: 2em;
    }

    包含选择器:
        外选择器 内选择器{属性:属性值;}

    table p {
        text-indent: 2em;
    }
