类选择器,即样式的分类选择器,可以根据不同的风格对某标签设置几种不同的CSS属性,或将整个风格分成几个类

    创建类选择器:
        元素.类选择器{属性:属性值;}
        层叠性
        .类选择器{属性:属性值}
        定义多个标签的类选择器

    引用类选择器:
        <标签 class="类选择器">
        类选择器不加前后缀
        <标签 class="类选择器1 类选择器2">
        引用多个类选择器于此标签时用空格隔开


    .setCenter {
        font-size: 10px;
        color: red;
        text-align: center;
    }

    <a class="setCenter">目录</a>


    id属性对单一元素进行识别

    id选择器的创建:
        元素#id选择器{属性:属性值;}
        #id选择器{属性:属性值;}
        更常见的用法,不针对某个元素

    id选择器的引用:
        <标签 id="id选择器">


    #setCenter {
        font-size: 10px;
        color: red;
        text-align: center;
    }

    <a id="setCenter">目录</a>
