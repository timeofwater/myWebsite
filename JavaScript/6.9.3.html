CSS JS 周期表 目标收集 
创建:
    document.createElement(nodeName);       创建元素节点
    document.createTextNode(text);          创建文本节点
    document.cloneNode(deep);               克隆节点
        deep = true     则递归复制所有子节点
           deep = false    (默认)只复制当前节点
    节点.appendChild(newNodeName)           插入子节点
    节点.insertBefore(newNodeName, node)    在node前插入子节点

删除:
    节点.removeChild(nodeName)              删除子节点

替换:
    节点.replaceChild(newNodeName, oldNodeName)
                                            替换子节点

属性节点:
    元素.getAttributeNode(attributeName)    获取属性节点
    document.createAttribute(attributeName) 创建属性节点
    元素.setAttributeNode(attributeName)    添加属性节点
