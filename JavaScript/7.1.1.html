jQuery 入口函数:

        $(document).ready(function(){
            // 执行代码
        });
    或者
        $(function(){
            // 执行代码
        });


    JavaScript 入口函数:

        window.onload = function () {
            // 执行代码
        }


    jQuery 入口函数与 JavaScript 入口函数的区别：

        jQuery 的入口函数是在 html 所有标签(DOM)都加载之后，就会去执行。
        JavaScript 的 window.onload 事件是等到所有内容，包括外部图片之类的文件加载完后，才会执行。
