jQuery 选择器基于元素的 id、类、类型、属性、属性值等"查找"（或选择）HTML 元素。
    它基于已经存在的 CSS 选择器，除此之外，它还有一些自定义的选择器。


    jQuery 中所有选择器都以美元符号开头：$()。


    示例：
    $(document).ready(function(){
        $("button").click(function(){
            $("p").hide();
        });
    });

    
    
    <p>p元素</p>



    #id 选择器


    示例：
    $(document).ready(function(){
        $("#b").click(function(){
            $("h3").hide();
        });
    });

    
    
    <h3>h3元素</h3>



    .class 选择器


    示例：
    $(document).ready(function(){
        $(".butt").click(function(){
            $("h4").hide();
        });
    });

    
    
    <h4>h4元素</h4>



    this 选择器


    示例：
    $(document).ready(function(){
        $("li").click(function(){
            $(this).hide();
        });

    子元素 选择器


    示例：
    $(document).ready(function(){
        $("div span").click(function(){
            $(this).hide();
        });
    });

    <span>
    <div>

    :first 和 :first-child 选择器
    div span:first 表示第一个div里的第一个span元素
    div span:first-child 表示每一个div里的第一个span元素


    示例：
    $(document).ready(function(){
        $(".ex1 div").click(function () {
            $(".ex1 div:first").hide();
        });
    });
    $(document).ready(function(){
        $(".ex2 div").click(function () {
            $(".ex2 div:first-child").hide();
        });
    });

    [属性] 选择器


    示例1：
    $(document).ready(function(){
        $("[href]").click(function(){
            $(this).hide();
        });
    });

    <a id="a1" href="#a1">

    $(document).ready(function(){
        $("a[href='#a2']").click(function(){
            $(this).hide();
        });
    });

    <a id="a2" href="#a2">
